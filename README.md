# 🌍 TripTour

A modern and responsive travel & tour booking platform built with React, TypeScript, Vite, and Tailwind CSS. TripTour helps users discover amazing destinations, explore tour packages, and plan memorable journeys through a clean and intuitive interface.

## ✨ Features

* 🏖️ Modern travel website UI
* 📱 Fully responsive design
* 🔍 Destination exploration
* 🎯 Tour package showcase
* 🖼️ Beautiful image galleries
* ⚡ Fast performance with Vite
* 🎨 Tailwind CSS styling
* 🌐 Cross-browser compatibility
* 🔥 Clean and maintainable code structure

## 🛠️ Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* React Router DOM

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

* Node.js (v18+ recommended)
* npm or yarn

### Installation

Clone the repository:

```bash
git clone https://github.com/IncridableAcuman/triptour.git
```

Navigate to the project directory:

```bash
cd triptour
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
